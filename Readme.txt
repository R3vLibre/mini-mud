A propos
========

Le projet 'mini-mud' est un atelier didactique pour la découverte de la
programmation C/C++, réalisé dans le cadre des cours à `ActivDesign
<http://activdesign.cc/>`_.


Ce document et les fichiers textes sont écrits dans le langage
`reStructuredText <http://docutils.sourceforge.net/rst.html>`_.


La documentation complète se trouve dans le répertoire ``doc``, y compris
une description de l'arborescence.

Ce *Readme* sera amené à être complété au fil de l'avancement du projet...
