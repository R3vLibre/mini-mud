Introduction
============

Mini-Mud est un projet didactique pour l'apprentissage du C++.
Il présente les fonctionnalités rudimentaires d'un MUD, dans une approche locale un joueur d'abord, client/serveur à travers le réseau ensuite.

Ce répertoire contient la documentation du jeu ainsi que les informations de réalisation.

A ce stade, le jeu est un prototype.
